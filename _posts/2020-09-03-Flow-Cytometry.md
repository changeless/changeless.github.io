# 概述
流式细胞术: 以流式细胞仪为检测手段，快速、准确、客观检测单个生物微粒的多项特性，并加以定量的技术，同时可以对特定群体加以分选。

# 原理

1. Cell suspension
2. Staining of membrane, cytoplasm, and nucleus
3. Cell sorting
4. Cell cultures, analysis, etc


## 提供三方面信息：
1. 前向角散射光（forward scatter，FSC）：相对大小
2. 侧向角散射光（Side scatter，SSC）：相对颗粒度
3. 荧光（Fluorescence): 相对荧光强度，代表相应标志物含量或水平

## 标记荧光抗体
###三原则
1. 必须能够被流式细胞仪上所配备的激光器所激发
2. 激发光谱必须在仪器上滤光片能够接受的合适范围内
3. 荧光素光谱的重叠应当尽量减少

## 对照设置
### 阴性对照的设置 - 确定细胞的非特异性荧光
#### 激光照射
1. 散射光信号(FSC,SSC)
2. 荧光信号(特异性荧光 / 自发性荧光 —— 空白对照（圈门）/非特异性荧光 ——同型对照（圈门）)

#### 非特异性结合: 抗体与细胞表面的Fc受体结合，或因化学键、生物键及分子之间吸引力造成的结合

### FMO对照（Fluorescence-minus-one control): 只缺少标记某一通道代表的荧光素偶联抗体，精确界定不同细胞亚群各自的这一通道的阴阳性界限，从而使流式分析结果更加精确。
#### 在一下情形下尤其重要
1. 应该较弱或分群不清的细胞群体
2. 在试剂优化时衡量荧光溢漏

### 阳性对照的设置 - 无需每次实验设置
#### 设置阳型对照的方法
1. 用肯定表达有该抗原的细胞来检测
2. 已经证明有效的/偶联其他荧光素的该抗原的抗体
#### 需要设置阳性对照的情况
1. 使用新的荧光素偶联抗体，该抗体以前从未使用过
2. 换用不同公司或不同批号的抗体
3. 使用储存时间较长的抗体时

## 数据分析和分选
### 电压设定 - 根据阴性对照调电压
1. 足够大的电压才能将阴性阳性分开
2. 如果电压过大，不但不能得到更好的结果，反而会加大背景
3. 补偿是受电压影响的，电压不同，补偿会不同

### 阈值设定 —— 影响纯度/效率
1. 设置阈值去除小颗粒，提高细胞比例，有助于提高分选效率；但并不代表小颗粒被物理的去除了，仪器“看不见”，它就会进入分选样本中。

### 补偿调节 —— 荧光补偿（荧光染料不是单色光，而是一段波长范围 —— 荧光补偿消除溢漏到相邻通道的荧光）
#### 荧光补偿流程
1. 制备单色对照：使用与实验中相同或相似的细胞
2. 设置仪器参数—PMT电压：基于阴性对照设置PMT电压
3. 分析单色对照结果：运行每个单染对照，从每个保存的数据文件中分析数据
4. 设置补偿矩阵：使用软件自动计算补偿矩阵

# 分选
## 分选的评价标准
1. 分选纯度（purity）：分选后目的细胞的比例
2. 分选得率（recovery）：目的细胞的回收程度
3. 分选速度（sorting rate）：分选的通量
4. 分选活性（viability）：所得细胞的存活率

## 影响分选结果的环节
1. 进样：温度，压力差（进样体积速率）
2. 细胞悬液：浓度、活性、粘连、目的细胞比例、类型、缓冲液成分
3. 液流断点：液滴延迟时间、断点稳定性、加电、卫星液滴、加电修正
4. 收集装置：温控，管式，板式
5. 鞘液流：成分，压力及稳定性
6. 检测与设置：灵敏度、圈门、分选模式
7. 芯片喷嘴：尺寸，最佳震荡频率，振幅
8. 偏转液滴：轨迹、落点
9. 收集液：温度、成分

## 基本步骤
### 样本制备
#### 目标
1. 细胞形态：单个、悬浮（不粘连、不沉降）
2. 高细胞活性：分选后细胞活性不可能比分选前高
3. 减少粘连体，无团块：易堵塞和影响液滴断裂；易影响纯度，影响回收率
4. 减少碎片/死细胞：碎片信号过多淹没电路，降低回收率
5. 不粘稠：影响液滴断裂
6. 合适浓度：一般提倡高浓度、低速进样（10^7 cells/ml，与细胞大小、粘性等性质密切相关）
### 操作要点
1. 去除团块和粘连：使用滤网过滤（35um，50um，70um）
2. 降低前稠度：细胞消化使用Accutase替代Trypsin+EDTA
3. 减少聚集体：样本中适当添加EDTA（1-5mM）和/或DNAse（20-200ug/mL）
4. 减少碎片：调整离心条件，选用尽量低的重力加速度
5. 低温制样、4度保存，避光
6. 如长时间分选，可制一批、分一批，缩短样本放置时间，并及时处理分选所得细胞
7. 若目的细胞比例过低，尽量预富集以提高比例（磁珠、密度梯度离心等）
### 常见问题及解决对策
1. 细胞密度低（制备过程中细胞损失）——增加离心时间，吸弃上请
2. 非特异荧光强（抗体不纯，死亡细胞多）——选择纯度高的抗体、用同型对照抗体或双标排除飞特异荧光
3. 碎片多（细胞死亡，机械损伤）——在细胞生长状态良好时测试，避免反复吹打
4. 细胞聚集（消化不完全、细胞自发聚集）——彻底消化，加入EDTA或DNAse
5. 荧光弱（灵敏度不够，抗体加入量不饱和，反应时间短）——改用生物素-亲和素，增肌抗体用量，延长反应时间
### 碎片对于分选质量的影响
1. 引起仪器信号处理电路的响应，碎片过多，会大量占用仪器处理资源，增加分选丢弃率，降低分选效率
2. 碎片附着积累可能引起喷嘴或进样管部分堵塞
3. 降低纯度：增加细胞到达液流断点的不可预测性
4. 低于阈值的碎片不被仪器检测，但仍存在于分选的液滴汇总
### 死细胞对于分选质量的影响
1. 产生碎片
2. 释放细胞质成分，影响活细胞功能
3. 释放基因组DNA，导致团块产生，增加细胞粘连性
4. 自发荧光增高而可能引起假阳性
5. 非特异吸附荧光抗体，增加假阳性
### 聚集体/团块对于分选质量的影响
1. 喷嘴堵塞
2. 进样管堵塞
3. 细胞进样速率波动
4. 造成液滴断点不稳定，降低分选效率
5. 造成测液流分散，降低回收率

## 分选模式选择
### 液流稳定：高纯度分选的前提 —— 断点位置稳定
### 分选模式选择
#### 冲突事件（conflicts）：两个细胞过于靠近以至于被包裹在一个液滴中；细胞在液滴中的位置靠近边缘时；下一滴含有不需要的细胞过于靠近时
#### 纯化模式
1. normal mode (no margin for error): 90-95%
2. Semi-purity mode (1/4 drop margin): 95-97
3. Purity Mode (1/2 drop margin): 97-99
4. Ultra-Purity Mode (1 drop margin): 99-99.9
#### 富集模式
1. Semi-yield：1/4 drop extension
2. Yield Mode：1/2 drop extension
3. Ultra-yield mode：3 drop sort
#### 单细胞模式
### 样品细胞粘连对分选纯度的影响
#### 正常运行的情况下，“粘连体”细胞是降低分选纯度的主要原因
#### 粘连体不可能百分之百完全排除
#### Double Discrimination Gating
1. FSC/BSC: 确定目标细胞群
2. FSC-H/FSC-W 或 FSC-H/FSC-A：确定单个细胞群
### 设门方式对分选纯度的影响
分选设门使用散点图而不是直方图，即使细胞只标记一种荧光素（可选用BSC或其他荧光通道作为第二坐标轴）
### 阈值设定影响分选的纯度、效率
1. 阈值设定对分选纯度的影响
2. 阈值以下的颗粒会被随机分布到所有液滴
3. 阈值过高会造成事实上的纯度降低
4. 阈值过低会造成冲突事件过多而直接影响得率

## 仪器设置——分选得率的影响因素
### 液滴延迟与液流稳定
### 分选模式（在得率优先的情况下，可以选择yield模式）
### 冲突事件发生几率对分选得率的影响
1. Event rate
2. 液滴大小（液滴生成频率，喷嘴尺寸）
3. 样本浓度
### 喷嘴选择	流速vs得率
#### 在频率一致的情况下（即采用同种喷嘴规格芯片时）
1. Higher yield - low event rate - more empty drops
2. Lower yield - high event rate - greater chance of coincidence
3. 提升得率的方法:降低Sample pressure, 降低样本浓度
#### 在上样速度一致的情况下（即相同event rate）
1. Higher yield - high frequency - small nozzle size - more empty drops
2. Lower yield - low frequency - large nozzle size - greater chance of coincidence
3. 提升得率的办法: 提高液滴生成频率, 即使用喷嘴更小的芯片

## 细胞活性的影响因素
### 减少样本数中死细胞比例
#### 样本制备
1. 酶消化时间不要过长
2. 细胞吹打温和
3. 离心尽量低速
####染料区分细胞死活
1. PI, 7-AAD, DAPI
2. Sytox Blue/ Green/ Organge/ Red
3. Via-Probe Green/ Via-Probe Red
4. FVS系列
5. Zombie系列
### 洁净的分选环节
### 适宜的喷嘴和鞘液压力
1. 脆弱细胞：低压+大喷嘴（包裹细胞的液滴越大，对细胞的冲击越小）
2. 做单细胞分选，尽量采用低压和大尺寸喷嘴
3. 对于直径大的细胞，采用大尺寸喷嘴
### 光滑的上样管路
上样管路内壁不平损伤细胞，并且可能形成断流而破坏液流稳定，需定期更换上样管路
### 正确的液滴偏转角度
1. 收集液加入更高浓度血清或直接用纯血清作为收集液（高浓度血清有利于保持细胞活性，但部分细胞可能对高浓度血清产生应激变化）
2. 收集液添加HEPES（10-50mM）（一般收集液稀释不得超过一倍）
3. 收集管预先包被收集液，并润洗管壁
### 严格的温控系统
1. 环境温度：低温恒温
2. 制样过程：保持在4-10度；分选保持在4度
3. 分选温控：低温保持细胞处于低代谢水平，抑制酶活性，可以减少凋亡和其他降低细胞活性或影响下油应用的细胞功能活动
